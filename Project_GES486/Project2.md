<b>Maryland Congressional Districts and Road Layouts<b>
  
**Project description:**
  
<b>This map was created using ArcMap software. I created this map for the citizens of Maryland to gain knowldege on which districts are Republican or Democrat. Each of the parties were categorized by the Marylands roads that separate the districts. The map signifies which disctricts, in the state of Maryland, voted for what political party.<b>
